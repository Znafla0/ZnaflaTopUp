Panduan Singkat Deploy ke GitHub Pages

  1. Buat akun GitHub (jika belum).
  2. Buat repository baru bernama nama-kamu.github.io (untuk user pages) atau repo apa saja untuk project pages.
  3. Upload file HTML ini (mis. index.html).
  4. Aktifkan GitHub Pages di Settings → Pages → pilih branch main dan folder root → Save.
  5. Dalam beberapa menit, website aktif di https://username.github.io.

Jika mau order lewat form ke email supplier, ganti nilai SUPPLIER_EMAIL di bagian script sesuai email supplier.
